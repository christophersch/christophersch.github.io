Small showcase of miscellaneous university projects, all made in Java using Gradle.

Project 1-2: Mission to Titan

Solar system simulation software, where the objective was to implement multiple differential equation solvers to create a somewhat realistic orbital physics engine, and to successfully plan a mission to Titan and back.

Main responsibilities:

  • Implementation of solar system physics
  • Graphical User Interface
  • Debugging tools
  • Data analytics tools

    • GUI
    Analytics tools

    Project 2-1: Dice Chess

    A chess game based on chance. Main project objectives were to program the base game and implement two different intelligent chess playing agents, one purely algorithmically and the other with machine learning.

    Main responsibilities:

  • Implementation of game loop
  • Board state notation (FEN encoding)
  • Bug fixing & code hygiene

    • GUI

    Project 2-2: Multi-Agent Surveillance

    Surveillance simulation software where an intelligent intruder agent tries to reach a target area while intelligent guard agents try to catch them.

    Main responsibilities:

  • Map Editor software
  • Genetic Algorithm fitness functions
  • Data analytics tools

    • Map editor software
    Neural network evolution visualization tool
